# PID 温度控制仿真系统

一款基于 Python + PyQt5 的温度控制仿真教学软件，支持5种控制策略、用户权限管理、实时波形显示与历史数据导出。

---

## 功能特性

### 控制策略（运行中可无缝切换）
| 编号 | 策略名称 | 说明 |
|------|----------|------|
| 1 | 普通PID（无限幅） | 无抗饱和，可观察积分饱和现象，教学演示用 |
| 2 | 单回路PID（带抗饱和） | 条件积分抗饱和，输出限幅 ±30 |
| 3 | 前馈 + 反馈控制 | 静态前馈补偿可测干扰，输出限幅 ±30 |
| 4 | 串级PID控制 | 外环控制最终输出，内环控制中间变量 |
| 5 | 串级 + 前馈控制 | 在串级基础上叠加前馈补偿 |

### 被控对象
- 双惯性环节串联：`G(s) = K / ((T₁s+1)(T₂s+1))`
- 反馈环节：一阶低通滤波器（模拟传感器动态）
- 干扰：方波干扰，可配置振幅和持续时间

### 用户管理
- 管理员 / 普通用户 两级权限
  - 普通用户：运行仿真、修改密码、查看历史曲线、导出历史 Excel
  - 管理员：额外拥有用户增删改、重置密码、切换角色
- 登录验证、修改密码、用户增删改
- 用户数据本地 JSON 存储，密码采用 SHA256 哈希

### 其他
- 实时波形：SV / PV / u / 干扰 / 误差，5条曲线
- 历史数据：按会话存储，支持查询和导出 Excel
- 手动 / 自动模式无扰动切换
- 完善的异常捕获机制，程序不闪退

## 使用说明

### 基本操作流程
1. 启动程序，使用默认账号登录
2. 在左侧面板选择**控制策略**
3. 设置 **SV 设定值**（范围 0 ~ 30）
4. 调整 **PID 参数**后点击「应用参数」
5. 点击「**开始仿真**」，右侧实时显示波形
6. 运行中可随时切换策略、修改参数（立即生效）

### 手动模式
- 勾选「手动模式」后，可直接输入控制量 u
- 切回自动模式时控制器自动跟踪当前输出，**无冲击**

### 施加干扰
1. 设置「振幅」和「持续时间」
2. 点击「施加方波干扰」
3. 按钮自动锁定并显示倒计时，干扰结束后自动恢复

### 历史数据
- 菜单 → 功能 → 历史曲线
- 选择记录后点击「加载」查看波形
- 支持鼠标缩放 / 平移
- 点击「导出Excel」保存完整数据

## 数据流示意

```
设定值 SV
    │
    ▼
控制策略管理器（PID / 串级 / 前馈）
    │  u
    ▼
被控对象（双惯性环节）──── + 干扰
    │  y
    ▼
反馈环节（一阶低通滤波）
    │  PV
    └──────────────────────── 反馈至控制器
```
