# PID温度控制仿真系统 打包说明

## 环境准备

- Python 3.10+
- Windows 10 / 11
- 安装依赖：`pip install PyQt5 pyqtgraph openpyxl pyinstaller`

## 一键打包

在项目根目录执行：

```bash
python build_exe.py
```

脚本会自动清理旧的构建缓存，并调用 PyInstaller 打包。

## 输出文件

打包成功后，单文件 EXE 位于：

```
dist\PID温度控制仿真系统.exe
```

该文件可独立运行，无需 Python 环境，Windows 免安装运行。

## 发布包结构建议

```
发布包/
├── PID温度控制仿真系统.exe
├── data/                    # 数据目录（程序运行时自动创建）
└── 使用说明.txt
```

## 常见问题

1. **打包失败提示目录非空**：`build_exe.py` 已配置自动清理 `dist` 目录，一般无需手动处理。
2. **运行 exe 提示缺少 DLL**：确保打包环境已安装完整的 PyQt5 和 pyqtgraph 依赖。
